
# Black Hole Simulation

Black Hole Simulation is a Vite + TypeScript web project that visualizes an interactive black hole scene in the browser. It is designed to demonstrate space-themed visuals, animation, and frontend development using modern web tools.

## Features

- Interactive black hole visualization
- Browser-based simulation
- Built with Vite and TypeScript
- Fast local development setup
- Ready for deployment on Netlify

## Tech Stack

- TypeScript
- Vite
- HTML
- CSS
- JavaScript

## Installation

Clone the repository:

```bash
git clone YOUR_REPOSITORY_URL
cd black-hole-simulation
````

Install dependencies:

```bash
npm install
```

## Usage

Run the development server:

```bash
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://localhost:5173/
```

## Build

Create a production build:

```bash
npm run build
```

The final build will be generated inside the `dist/` folder.

## Deployment

For Netlify:

```text
Build command: npm run build
Publish directory: dist
```
